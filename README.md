## Featured Projects

### [ThinkSpeed](https://github.com/HirokiTakano/ThinkSpeed)

A lightweight outliner and thinking tool for quickly capturing ideas as bullet points and organizing them into a structured hierarchy.

- Built with Next.js, React, TypeScript, Tailwind CSS, and Tiptap
- Local-first data storage using localStorage
- Supports folders, files, search, custom shortcuts, themes, image paste, Markdown copy, and JSON backup
- Deployed with AWS Amplify

### [PCDetailCheck](https://github.com/HirokiTakano/PCDetailCheck)

A Windows batch-based utility for checking detailed PC information.

- Displays OS, CPU, memory, disk, GPU, and network information
- Uses PowerShell to collect system information
- Designed for local execution without uploading data to external servers

### [Portfolio Site](https://main.d39h1h78ufdmo2.amplifyapp.com/)

My personal portfolio site for introducing my projects, skills, and learning journey.

- Built with Next.js, TypeScript, Tailwind CSS, Framer Motion, and lucide-react
